# データ一覧

| データ名 | 説明 | 管理すべき情報 | 備考 |
|---------|------|---------------|------|
| 書籍情報 | ユーザーが登録する書籍 | タイトル、著者、ISBN（ユニーク）、出版日、説明（任意）、画像URL（任意）、作成者ID 等の基本情報と登録者情報 | `books`テーブル |
| ジャンルマスタ | 書籍分類 | ジャンル名（ユニーク） | `genres`テーブル |
| 書籍×ジャンル | 多対多紐付け | 書籍IDとジャンルIDの組み合わせ | `book_genre`テーブル |
| レビュー | 書籍へのレビュー | 投稿者、対象書籍、評価値（1〜5）、コメント | `reviews`テーブル |
| お気に入り | ユーザーのお気に入り書籍 | ユーザーIDと書籍IDの組み合わせ | `favorites`テーブル |
| いいね | レビューへのいいね | ユーザーIDとレビューIDの組み合わせ | `review_likes`テーブル |
| ユーザー | アプリケーション利用者 | 名前、メールアドレス（ユニーク）、パスワード（ハッシュ化）等 | `users`テーブル |
| 認証補助 | パスワードリセットトークン等 | パスワードリセット、パーソナルアクセストークン、失敗ジョブ | Laravel標準マイグレーション |
| ★ 書籍情報 | 【変更】 | `isbn`: NOT NULL → nullable に変更<br>`published_date`: NOT NULL → nullable に変更 | |
| ★ 読書計画 | ユーザーが書籍ごとに「いつまでに読むか」を計画するレコード | 計画者、対象書籍、期日、計画状態、関連する日時情報 | （テーブル名未定） |
| ★ 通知 | 読書計画リマインダーの通知レコード | 受信者、通知種別、通知内容、既読状態 | Laravel標準の通知機能を使用する |
```
