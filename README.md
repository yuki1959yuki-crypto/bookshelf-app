# 書籍レビューAPI・Web管理システム

書籍の管理業務を行うWebアプリケーションと、外部システム連携を可能にする公開APIを統合したシステムです。
管理者は書籍情報のCRUD操作（登録・編集・削除）を行い、一般ユーザーは書籍の検索・閲覧・レビュー投稿を通じて書籍情報の活用を行うことができます。
